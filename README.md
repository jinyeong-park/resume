# HTML Resume builder
Let's build your resume site with AI


### Simple prompt
copy and paste this propmpt into your favorite AI Tools (ChatGPT, Claude, Gemini, etc)
```
Based on whatever you know about me. Help me create a well designed futuristic resume for myself. The resume should be in html, css, and javascript all in one file.
```
